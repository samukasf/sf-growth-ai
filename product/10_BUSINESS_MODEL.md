# Business Model — SF Growth AI

## Visão geral

O SF Growth AI opera em **quatro linhas de receita** complementares, todas servindo à mesma promessa: **administrar empresas com Supercérebro**.

---

## Linha 1 — Agency Platform

**Para quem:** Agências de marketing, comunicação, consultoria e serviços B2B.

| Elemento | Descrição |
|----------|-----------|
| **Proposta** | Infraestrutura cognitiva white-label para operar múltiplos clientes |
| **Modelo** | SaaS por agência + fee por cliente onboarded |
| **Valor** | Escala sem headcount; diferenciação competitiva |
| **Produto** | Agency Workspace, onboarding, multi-tenant |
| **Lighthouse** | Influence Publicidade |

**Tiers sugeridos:**

| Tier | Clientes | Preço indicativo |
|------|----------|------------------|
| Agency Starter | até 10 | Base mensal |
| Agency Pro | até 50 | Base + per client |
| Agency Enterprise | ilimitado | Custom |

---

## Linha 2 — Enterprise

**Para quem:** Médias empresas que operam diretamente (sem agência).

| Elemento | Descrição |
|----------|-----------|
| **Proposta** | Supercérebro dedicado para a própria empresa |
| **Modelo** | SaaS por empresa + módulos premium |
| **Valor** | CEO digital + conselho + assessment contínuo |
| **Produto** | Executive Workspace, Company Brain, Council, CEO |

**Tiers sugeridos:**

| Tier | Capacidades |
|------|-------------|
| Enterprise Starter | Brain + CEO + Assessment |
| Enterprise Pro | + Council + Projects + Opportunities |
| Enterprise OS | + Software Factory + Autonomia supervisionada |

---

## Linha 3 — Marketplace

**Para quem:** Empresas, agências e parceiros certificados.

| Elemento | Descrição |
|----------|-----------|
| **Proposta** | Ecossistema de integrações, templates, metodologias e serviços |
| **Modelo** | Comissão sobre transações + listing fee |
| **Valor** | Extensibilidade sem fragmentar o Supercérebro |
| **Produto** | Integrações (GA, Meta, CRM), templates, playbooks |

**Exemplos de oferta:**

- Conectores certificados (Google Analytics, Meta Ads, Search Console)
- Templates de assessment por indústria
- Playbooks executivos (Samuel Standard certified)
- Serviços de parceiros (implementação, consultoria)

---

## Linha 4 — Software Factory

**Para quem:** Empresas e agências que transformam oportunidades em software.

| Elemento | Descrição |
|----------|-----------|
| **Proposta** | Da oportunidade detectada ao software entregue |
| **Modelo** | Project fee + revenue share + manutenção |
| **Valor** | Oportunidade → projeto → software → ROI mensurável |
| **Produto** | Software Factory pipeline |

**Fluxo comercial:**

```
Opportunity detectada → Projeto aprovado → Software Factory → Deploy → ROI tracking
```

## Go-to-market

| Fase | Foco | Receita |
|------|------|---------|
| **Alpha** | Lighthouse (Influence) — validação | Zero / pilot |
| **Beta** | 3–5 agências piloto | Early adopter pricing |
| **Version 1** | Agency Platform comercial | Recurring |
| **2028+** | Enterprise + Marketplace | Diversificada |

---

## Unit economics (meta)

| Métrica | Target |
|---------|--------|
| CAC (agência) | < 3 meses de MRR |
| LTV/CAC | > 5x |
| Churn mensal | < 3% |
| Clientes por agência | > 15 (média) |
| Time to first value | < 15 min (onboarding) |

---

## Princípio comercial

> Vendemos **capacidade cognitiva**, não licenças de software.

O cliente paga pelo Supercérebro que administra empresas — não por seats ou features.
