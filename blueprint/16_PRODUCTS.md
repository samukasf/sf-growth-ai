# Products

## Objetivo

Descrever o portfólio de produtos do SF Growth AI e como cada um se posiciona no ecossistema.

## Responsabilidades

- Definir produtos atuais e planejados
- Clarificar público-alvo de cada produto
- Mapear produtos para módulos técnicos
- Orientar go-to-market e packaging

## Componentes

| Produto | Descrição | Status |
|---------|-----------|--------|
| **Samuel AI** | CEO digital + Conselho Executivo | Ativo (demo) |
| **Executive Workspace** | Interface principal do produto | Ativo |
| **Organization Platform** | Multiusuário, permissões, auditoria | Core pronto |
| **Executive Engines Suite** | Knowledge, Learning, Wisdom, etc. | Core pronto |
| **Software Factory** | Geração de software | Planejado |
| **Marketplace** | Soluções e integrações | Planejado |
| **Partner Platform** | Canal de parceiros | Planejado |
| **Agency Platform** | Multi-cliente para agências | Planejado |

## Fluxo

```
Prospect → Demo Samuel AI
      ↓
Piloto (organização configurada)
      ↓
Assinatura (tier por produto)
      ↓
Expansão (engines, factory, marketplace)
```
